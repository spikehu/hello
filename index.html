np.zeros()

Posted on 2021-04-18 In python语法

np.zeros(shape, dtype=None, order=’C’)

官方注释

返回给定维度和数据类型的数组，数据为0，默认类型是float

参数

zeros(shape, dtype=float, order='C')
    Return a new array of given shape and type, filled with zeros.    
    Parameters
    ----------
    shape : int or tuple of ints
        Shape of the new array, e.g., ``(2, 3)`` or ``2``.
    dtype : data-type, optional
        The desired data-type for the array, e.g., `numpy.int8`.  Default is
        `numpy.float64`.
    order : {'C', 'F'}, optional, default: 'C'
        Whether to store multi-dimensional data in row-major
        (C-style) or column-major (Fortran-style) order in
        memory.

shape:指定数组的维度
dtype:数据类型，默认float
order:表示行优先还是列优先，默认是C行优先，F是列优先

    Returns
    -------
    out : ndarray
        Array of zeros with the given shape, dtype, and order.

    See Also
    --------
    zeros_like : Return an array of zeros with shape and type of input.
    empty : Return a new uninitialized array.
    ones : Return a new array setting values to one.
    full : Return a new array of given shape filled with value.

例子

    Examples
    --------
    >>> np.zeros(5)
    array([ 0.,  0.,  0.,  0.,  0.])

    >>> np.zeros((5,), dtype=int)
    array([0, 0, 0, 0, 0])

    >>> np.zeros((2, 1))
    array([[ 0.],
           [ 0.]])

    >>> s = (2,2)
    >>> np.zeros(s)
    array([[ 0.,  0.],
           [ 0.,  0.]])

    >>> np.zeros((2,), dtype=[('x', 'i4'), ('y', 'i4')]) # custom dtype
    array([(0, 0), (0, 0)],
          dtype=[('x', '<i4'), ('y', '<i4')])

dtype=[(‘x’, ‘i4’), (‘y’, ‘float’)]表示可以用array[‘x]访问每组第一个元素，array[‘y’]就是第二个了。’i4’,表示每组第一个是int类型，’float’则是每组第二个是float类型。

np.squeeze()

官方注释

np.squeeze(a,axis)是将数组中一维的维度去掉

Remove single-dimensional entries from the shape of an array.

Parameters
----------
a : array_like
    Input data.
axis : None or int or tuple of ints, optional
    .. versionadded:: 1.7.0

    Selects a subset of the single-dimensional entries in the
    shape. If an axis is selected with shape entry greater than
    one, an error is raised.

Returns
-------
squeezed : ndarray
    The input array, but with all or a subset of the
    dimensions of length 1 removed. This is always `a` itself
    or a view into `a`.

Raises
------
ValueError
    If `axis` is not `None`, and an axis being squeezed is not of length 1

See Also
--------
expand_dims : The inverse operation, adding singleton dimensions
reshape : Insert, remove, and combine dimensions, and resize existing ones

传入数组a,如果不指定去掉的维数（维数必须是1，否则报错），则去掉所有维度为1的维，返回的依然是一个数组。
